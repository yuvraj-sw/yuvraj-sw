# 💫 About Me:
I am an AI and Machine Learning student focused on building systems where intelligent software meets practical, real-world hardware. My work centers on high-performance logic, computer vision, and reactive automation, bridging the gap between algorithmic models and physical devices. On the software side, I write clean, scalable code primarily using Python and C++, while handling embedded microcontrollers to drive automation on the hardware side. I approach engineering with a logic-first mindset, always aiming for clean system architecture and efficient execution. Right now, I am looking to connect and collaborate on innovative open-source AI tools and technical projects.
